### **1. Input Ingestion**

Reads the dataset of journal entries:

* `entry_id`
* `date`
* `text`

Dataset example:
Your dataset: dataset.json

### **2. LLM Annotation**

The Groq LLM assigns:

* `theme_label` (e.g., *Work Stress*, *Meaning & Values Reflection*)
* `emotional_intensity` (1â€“5)

### **3. Phase Detection**

Consecutive entries with the same theme become a **phase**, including:

* theme label
* start/end dates
* representative entries
* average emotional intensity ("confidence")

### **4. Micro-practices**

The LLM generates:

* Per-phase micro-practices
* Weekly micro-practices (ISO week grouping)

### **5. Insight Generation**

Produces a **1â€“2 sentence summary** of emotional evolution.

### **6. Timeline Visualization**

A matplotlib chart showing:

* X-axis: dates
* Y-axis: themes
* Point color: emotional intensity
* Phase bars
